Lab1: Intro to Logic Simulation

DESCRIPTION:

This lab uses a Multimedia Logic program to display:
Part A) The hex number of bits active represented by the input switches

Part B) The output of a truth table using a circuit.

Part C) The output of a different truth table solved using 
SoP and Boolean Algebra using a normal circuit,
a circuit exclusively constructed out of NAND gates,
and a circuit exclusively constructed out of NOR gates.
