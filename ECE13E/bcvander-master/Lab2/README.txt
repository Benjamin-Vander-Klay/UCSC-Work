Lab 2: Simple Data Path

-------------------------------------------------------------------------------

DESCRIPTION

In this Lab we used Multimedia Logic Program to create a simple date path
consisting of four 4-bit registers (made from D-Latch flip flops), 
an ALU (which performs a bitwise right rotation), 
and user input from switches and a 16 number keypad.

-------------------------------------------------------------------------------

FILES 
* Lab2.lgi
	This file is the MML file containing the entire lab 

-------------------------------------------------------------------------------

INSTRUCTIONS 

Boot up the Lab2.lgi file on a Multimedia Logic program and start the program
using the green play button and manipulate the switches and keypad to test the
ALU's ability to rotate bytes.
